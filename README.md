# Graphics_Lab

OpenGL 기반 렌더링 파이프라인 학습 및 그래픽스 기초 실습

## Environment

- **Framework**: openFrameworks v0.10.0 (vs2017)
- **IDE**: Visual Studio 2017+
- **Language**: C++ / GLSL

## Projects

| 폴더 | 설명 |
|------|------|
| OpenGLProject | 렌더링 파이프라인 기초 (indices, texture, tint, bright) |

## Setup

1. [openFrameworks v0.10.0](https://openframeworks.cc/download/) 다운로드
2. `apps/myApps/` 경로에 이 repo를 clone
3. Visual Studio에서 `.sln` 열어서 빌드
